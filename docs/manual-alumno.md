# Manual del alumno

## Acceso

Ingresar desde el dominio del alumno, por ejemplo:

```text
https://alumno.example.com
```

Usar correo y contrasena asignados por la institucion.

## Inicio

La pantalla principal muestra examenes disponibles. Un examen puede no aparecer si:

- Aun no inicia su periodo de disponibilidad.
- Ya vencio.
- Ya se alcanzo el maximo de intentos.
- La asignacion fue revocada.

## Antes del examen

Recomendaciones:

- Usar una conexion estable.
- Cerrar aplicaciones innecesarias.
- Tener cargador conectado si usa laptop.
- No abrir otras pestanas durante el examen.
- Leer instrucciones antes de iniciar.

## Durante el examen

El examen puede incluir:

- Temporizador general.
- Temporizador por pregunta.
- Guardado automatico.
- Avance automatico al terminar el tiempo de pregunta.
- Finalizacion automatica al terminar el tiempo general.

Las respuestas tambien pueden guardarse manualmente con el boton `Guardar`.

## Modo seguro

Para responder se requiere pantalla completa.

El sistema registra eventos como:

- Salida de pantalla completa.
- Cambio de pestana.
- Perdida de foco de ventana.
- Intentos de copiar, pegar o cortar.
- Clic derecho.
- Atajos bloqueados.
- Intentos de impresion.

Estos eventos son revisados por administradores o revisores academicos.

## Finalizar examen

Al completar las preguntas, usar `Finalizar examen`.

Si el tiempo general termina, el sistema finaliza automaticamente y envia lo guardado hasta ese momento.

## Resultados

Los resultados pueden mostrarse inmediatamente o quedar pendientes de revision, segun la configuracion del examen.

Si hay preguntas de desarrollo o casos clinicos, el resultado puede tardar hasta que un revisor lo publique.

## Problemas comunes

No puedo entrar:

- Verificar correo y contrasena.
- Confirmar con administracion que el usuario esta activo.

No aparece el examen:

- Revisar fecha y hora de disponibilidad.
- Confirmar que el examen fue asignado.

Se cerro pantalla completa:

- Activar nuevamente pantalla completa para continuar.

Se perdio conexion:

- No cerrar la pantalla.
- Esperar reconexion si es posible.
- Informar al administrador si no puede continuar.
